Site Institucional e Agendamento para Psicóloga

📋 Resumo

Site institucional e de agendamento para psicóloga online. Desenvolvido com HTML5, CSS3 e JavaScript puro (Vanilla JS), sem dependência de frameworks. Conta com sistema de agendamento interativo com calendário dinâmico, bloqueio de horários por sessão, integração com WhatsApp e layout totalmente responsivo para mobile.

✨ Funcionalidades Desenvolvidas

  • Header fixo com navegação suave entre seções (scroll behavior)
  • Menu hambúrguer responsivo para mobile
  • Destaque automático do link de navegação conforme o scroll da página
  • Modal de agendamento com abertura/fechamento por clique, ESC e clique fora
  • Calendário interativo com seleção de mês → dia → horário em etapas
  • Bloqueio de horários já agendados usando sessionStorage
  • Integração com WhatsApp — ao confirmar, monta a mensagem automaticamente e redireciona para o wa.me
  • Validação de formulário com feedbacks de alerta ao usuário
  • Botão flutuante do WhatsApp na tela
  • SEO básico com meta description e verificação do Google Search Console
  • Favicon configurado para desktop e mobile (manifest, apple-touch-icon)

🛠️ Tecnologias Utilizadas

HTML5 — Estrutura e semântica do site
CSS3 — Estilização, responsividade e layout
JavaScript (Vanilla JS) — Toda a interatividade, sem frameworks
Font Awesome 6 — Ícones (via CDN)

Sistema de Relatorio Tecnico

📋 Resumo

Aplicação web progressiva (PWA) desenvolvida em React + JavaScript para geração de relatórios técnicos de engenharia civil. Permite criar, editar e exportar relatórios de vistoria predial em PDF com suporte a fotos lado a lado, editor de imagens embutido, parecer técnico automatizado e histórico organizado por pastas. Funciona offline e usa Firebase Hosting apenas para servir os ficheiros estáticos.

✨ Funcionalidades principais

  • 3 tipos de relatório: Pessoa Física, Condomínio, Empresa (completo/técnico)
  • Formulário dinâmico com campos que aparecem/ocultam conforme o tipo selecionado
  • Cômodos e elementos em estrutura em árvore (cômodo → elemento → fotos)
  • Upload de fotos (galeria ou câmera) com compressão automática (~300KB)
  • Editor de fotos embutido (linhas, formas, texto, ajustes, desfazer)
  • Fotos lado a lado no PDF para comparativo (Situação × Termográfica)
  • Parecer técnico com modelos pré-definidos e preenchimento automático
  • Geração automática de texto para resumo e condições com base nos dados
  • Histórico de relatórios organizado por mês e pastas do usuário
  • Assinatura digital via canvas + upload, salva no relatório
  • Suporte multi-país: Brasil e Portugal
  • Offline / Client-side: tudo salvo em localStorage; hospedagem estática no Firebase Hosting
  • Auto-save de rascunhos no localStorage
  • Responsivo para mobile e desktop

🛠️ Tecnologias usadas

Linguagem: JavaScript (ES6+)
Framework: React 18 (Vite)
PDF: jsPDF (geração client-side)
Edição de imagens: Canvas API + browser-image-compression
Armazenamento: localStorage do navegador
Hospedagem: Firebase Hosting
Estilização: CSS puro (components.css)

Finance Hub

📋 Resumo

Aplicação web full-stack de controle financeiro pessoal com dashboard interativo, gestão de carteiras, transações, orçamentos e monitoramento do mercado de ações e câmbio em tempo real.

✨ Funcionalidades Principais

  • Autenticação completa com cadastro, login, confirmação de email e admin
  • Carteiras financeiras para conta bancária, dinheiro, investimento e cartão de crédito
  • Transações com categorização, filtros e transferências
  • Orçamentos por categoria com progresso visual
  • Dashboard com resumo financeiro, cards e notícias do mercado
  • Mercado de ações com cotações em tempo real e gráficos de linha
  • Conversor de câmbio entre 10 moedas com taxas atualizadas
  • Relatórios de despesas e receitas por categoria
  • Painel Admin para gestão de usuários
  • Tema claro/escuro e suporte a múltiplos idiomas

🛠️ Stack Tecnológica

Frontend: Vue 3 + Nuxt 3 (SSR)
Linguagem: TypeScript 5.7
Estilização: Tailwind CSS v4 + Nuxt UI v3
Banco de Dados: Supabase PostgreSQL + Auth
APIs: Brapi.dev, Frankfurter API
Build: Vite via Nuxt 3

🏗️ Arquitetura

Monolito full-stack com Nuxt 3, usando file-based routing em app/pages/, APIs REST em server/api/, Supabase com RLS e estado global via useState().

ConstruTrack

📋 Resumo

ConstruTrack é uma plataforma de gestão de projetos de construção civil que reúne tarefas, equipes, controle de materiais, prazos e relatórios em um único painel. Foi desenvolvida para dar visibilidade ao canteiro de obras, melhorar a comunicação entre equipes e agilizar a entrega de projetos.

✨ Funcionalidades principais

  • Dashboard de obras com progresso, custos e prioridades em tempo real
  • Gestão de tarefas e atividades com atribuição para equipes de campo
  • Controle de materiais e consumo por etapa de obra
  • Relatórios gerenciais exportáveis para PDF e planilhas
  • Agenda de obra com marcos, fases e prazos
  • Notificações e alertas para atrasos e aprovações pendentes
  • Autenticação segura com usuários e permissões
  • Responsivo para desktop e mobile

🛠️ Tecnologias utilizadas

Front-end: React + Tailwind CSS
Back-end: Node.js + Express
Banco de dados: SQL Server
Infraestrutura: AWS S3 + Azure DevOps
Arquitetura: Microserviços e APIs RESTful
